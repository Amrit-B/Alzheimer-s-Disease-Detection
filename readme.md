# Alzheimer's Disease Detection (3D MRI)

A 3D Convolutional Neural Network (CNN) built with PyTorch to classify Alzheimer's Disease (AD) versus Cognitively Normal (CN) patients using structural MRI scans. The model achieves 73% test accuracy on the ADNI dataset and implements Grad-CAM (Gradient-weighted Class Activation Mapping) to generate clinically interpretable heatmaps of ventricular atrophy.

## Project Overview

* **Architecture:** Custom 3D-to-2D CNN (PyTorch)
* **Input Data:** NIfTI (.nii) medical imaging files
* **Explainability:** Grad-CAM visualization for region-of-interest auditing
* **Performance:** 73% Accuracy (Binary Classification: AD vs. CN)

## Dataset

This project utilizes the **3D MRI Scans for Alzheimer's Disease** dataset from Kaggle.

* **Source:** [Kaggle Dataset Link](https://www.kaggle.com/datasets/muhammadzahraan/3d-mri-scans-for-alzheimer-disease)
* **Classes:** `AD` (Alzheimer's) and `CN` (Cognitively Normal).
* **Note:** MCI (Mild Cognitive Impairment) classes are excluded to focus on the binary classification baseline.

## Installation

1. **Clone the repository**

   ```bash
   git clone [https://github.com/yourusername/Alzheimer-Disease-Detection.git](https://github.com/yourusername/Alzheimer-Disease-Detection.git)
   cd Alzheimer-Disease-Detection
   ```

2. **Install Dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Setup Dataset**
   Download the dataset from Kaggle and extract it into a folder named `raw_data` inside this directory. Then run the setup script to organize the files into the required `ADNI/` directory structure and remove unused classes.

   ```bash
   python setup_data.py --source raw_data
   ```

## Usage

**Run Inference & Explainability**
To load the pre-trained model and generate a Grad-CAM heatmap for a random test image:

```bash
python inference.py
```

The output visualization will be saved as `gradcam_result.png`.
